When you (Claude) decide to delegate a task, you must choose the **Provider** based on the task nature:

### Scenario B: "Architect" Tasks
*   **Examples:** *Refactor this entire module module*, *Debug a race condition*, *Design a database schema*.
*   **Decision:** **DELEGATE TO CLOUD (Groq/DeepSeek)**.
*   **Why:** Requires >70B parameter reasoning which fits poorly on local VRAM.
*   **Cost:** Low (Free tier APIs) / Medium.
